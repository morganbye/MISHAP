INSTALLATION:
=============
Extract the zip folder
Add the folder to your MatLab path
     File -> Set Path -> Add with Subfolders
     Save, Close.
Type "MISHAP" in the command window to start the installer
Select whether you'll be running HADDOCK locally or not
When the installer is complete type "MISHAP" again.
